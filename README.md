# PaiFu 牌庫

Pokemon TCG app mainly built for serious Traditional Chinese players.

- Cards and UI in Traditional Chinese
- Browse and search cards across all HK sets
- Track your collection across all variants
- Evolution line view
- Deck builder with Pokécabook import — see what you're missing and export for tournament registration
- Standard legality checker
- Trade hosting — list your haves and wants, match with other players
- Deck analytics — damage output, prize mapping, consistency and disruption counts
